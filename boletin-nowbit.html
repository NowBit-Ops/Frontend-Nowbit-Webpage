¿QUÉ ES ES|QL?

ES|QL, o Elasticsearch Query Language, es un lenguaje de consulta innovador
que simplifica las investigaciones de datos y acelera la respuesta de las
consultas para un procesamiento de datos eficiente, sin problemas y rápido.
Con ES|QL, puedes buscar, agregar, calcular, transformar y visualizar todo
desde una sola ventana.

CARACTERÍSTICAS TÉCNICAS

MOTOR DE CONSULTA DINÁMICO

ES|QL te permite buscar, agregar, calcular, transformar y visualizar todo desde
una sola ventana para una mayor precisión, una investigación de datos
simplificada y una experiencia de consulta unificada.

NUEVAS CAPACIDADES DE BÚSQUEDA:

ES|QL ofrece nuevas capacidades, como búsqueda y uniones, lo que te permite
buscar, agregar, calcular y realizar transformaciones de datos con una sola
consulta.

EXPERIENCIA DE USUARIO SIMPLIFICADA:

Agregar fácilmente nuevas etapas para refinar tus resultados, remapear
archivos, calcular nuevos elementos y más.

ALERTAS PRECISAS:

Mejora la precisión de la detección con ES|QL. Puedes revisar tendencias
significativas sobre incidentes aislados, reducir falsos positivos y obtener
notificaciones más accionables.

CONSULTAS MÁS RÁPIDAS:

Con ES|QL, puedes ejecutar búsquedas y agregaciones en múltiples etapas
simultáneamente para una mayor velocidad y eficiencia.

¿CÓMO PUEDE AYUDAR ES|QL
A TU ORGANIZACIÓN?

Con ES|QL, puedes usar una sola consulta para analizar registros, métricas,
rastreos y datos de perfilado, además de identificar cuellos de botella de
rendimiento y problemas del sistema, reduciendo el tiempo de resolución.
Cuando combinas ES|QL, con Machine Learning de Elastic y AIOps, puedes
identificar tendencias, aislar incidentes, reducir falsos positivos y
proporcionar notificaciones más accionables para mejorar la precisión de la
detección.
